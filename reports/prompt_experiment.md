# Prompt Experiment

## Câu hỏi test

RAG là gì và tại sao dùng RAG?

---

## Prompt V1

Prompt:

```text
Giải thích RAG là gì.
```

Kết quả:

```text
Dán kết quả Gemini trả lời ở đây.
```

Nhận xét:

Prompt này đơn giản nhưng có thể khiến AI trả lời chung chung.

## Prompt V3

Prompt:

```text
Bạn là AI Engineer mentor.

Hãy giải thích RAG cho người mới bắt đầu theo format:
1. RAG là gì?
2. Vì sao cần RAG?
3. Ví dụ thực tế
4. Hạn chế của RAG

Trả lời ngắn gọn, dễ hiểu, bằng tiếng Việt.
```

Kết quả:

```text
Dán kết quả Gemini trả lời ở đây.
```

Nhận xét:

Prompt này tốt nhất vì có vai trò, format và giới hạn rõ ràng.
