# AI RAG Document Assistant

## Overview

This is a beginner-friendly mini project for building a document Q&A assistant using Python and Gemini API.

## Day 1 Progress

- Set up Python project
- Connected Gemini API
- Created basic LLM calling script
- Created prompt experiment report
- Loaded text document
- Split document into chunks
- Saved chunks to JSON

## Tech Stack

- Python
- Gemini API
- google-genai
- python-dotenv
- JSON

```text
data/
src/

README.md

```

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run basic LLM script:

```bash
python src/ask_llm.py
```

Run chunking script:

```bash
python src/chunking.py
```
